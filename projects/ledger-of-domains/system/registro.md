# Domain: Registro (Oliver)

## Copy conditions — in order

| # | Condition | Without it → |
|---|---|---|
| **0** | **See the active Odem of the Domain** — base condition | Copy impossible. Passive signature accumulates but does not execute |
| **1** | Target Domain **active** (not latent) | Signature only — pattern unreadable |
| **2** | **Sustained exposure** seeing the pattern | Crude or failed copy. Time varies with complexity |
| **3** | **Deliberate attention** — reading the structure, not just observing | Involuntary copy, no control |
| **4** | **Physical cost of activation** | First copy of each Domain = high cost. Subsequent copies of the same: lower |

## Degradation

- An unpracticed copy loses fidelity over time. Does not disappear — becomes cruder.
- Cannot "recharge" from passive signature — must see the active Domain again.

## Passive memory of the Registro

Accumulates signature readings without Oliver deciding. If accumulation surpasses the threshold of condition 2 and the Domain activates nearby, it can trigger without condition 3 being met — involuntary, malformed copy.

## Absolute limit: blindfold = total shutdown

Registro activates through vision. Without sight: no reading, no copy. **Minimum effective AMAT containment:** blindfold in transport + unsupervised periods.

## Movement reading (seed Cap 14)

Oliver can read the structure of a physical movement with the same mechanics as reading a Domain — layers, weight origin, sequence, anchor point. Reading ≠ reproducing; they are separate steps. *Does not require active Odem* — operates on pattern in physical information.

## Long-term cost — cost arc (Caps 24–31)

The Registro has been continuously active for weeks. The threshold between passive and active reading moves on its own. The distinction between "Oliver perceiving" and "what the Registro processes" becomes progressively less clear.

**Cost vector:** identity first (Arc 3) → memory → body.

**Solution:** learn **selective attention** — seeing without reading. Prerequisite: knowing where one's own Odem is (seed planted by Park in Cap 23).

**Kaspar connection:** his signature is "composite, superimposed" because he has been in that gradient for 40 years. Oliver is in week seventeen. Kaspar is what lies at the end of the curve without intervention.

## Oliver's progression path

See Odem → read behavior → copy foreign Domain → cost per copy → degradation without practice → long-term cost (identity → memory → body).
